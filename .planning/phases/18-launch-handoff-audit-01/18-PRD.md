# Phase 18 PRD — Launch Handoff Audit 01

## Goal

Audit the live Ghosthand repo to determine whether a clean app launch/open primitive is already present, only partially present and under-expressive, or genuinely absent.

## Scope

- inspect live runtime/API/catalog/docs for app launch/open behavior
- separate live substrate truth from historical or stale references
- produce one final classification
- define the narrowest next implementation target only if the audit justifies it

## Out of Scope

- no new endpoint implementation
- no generic app-control architecture
- no skill-repo work
- no broad scenario expansion
