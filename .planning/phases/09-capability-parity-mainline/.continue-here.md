- MediaProjection lifecycle: Activity consent → GhosthandServiceRegistry singleton → LocalApiServer → StateCoordinator → MediaProjectionProvider
  GhosthandServiceRegistry avoids private companion visibility issues when accessed from MainActivity
- Display densityDpi from context.resources.displayMetrics.densityDpi (not Display.densityDpi) for API compatibility
